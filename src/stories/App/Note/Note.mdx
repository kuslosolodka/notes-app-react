# Definition

The Note component is a fundamental building block of the Notelist component, allowing users to view and manage individual notes within a collection.

It serves as a container for displaying specific pieces of data related to each note, such as title, content, date, and any other relevant information.

## Usage

Note displaying data such as `category`, `content`, `createdAt`, `date`, `name`, `updatedAt`
