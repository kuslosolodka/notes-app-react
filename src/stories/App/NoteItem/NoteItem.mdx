# Definition

The Note Item Component is a reusable UI element meant to represent a single note within a larger collection of notes, typically displayed in a list or grid format.

It provides a consistent and organized way to showcase individual notes with relevant details.

## Usage

Note Item displaying note details: `category`, `name`, `content`, `createdAt`, `updatedAt`.
