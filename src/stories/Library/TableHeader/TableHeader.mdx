# Definition

The Table Header Component plays a crucial role in visually organizing and labeling the columns of a table.

It provides users with a clear understanding of the data presented in each column.

## Usage

Table header using in table for displaying data table heading: `category`, `name`, `content`, `date`, `createdAt`, `updatedAt`.
