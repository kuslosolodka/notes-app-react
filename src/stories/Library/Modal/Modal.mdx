# Definition

A modal is a user interface component that appears on top of the main content to capture user attention and provide a focused interaction.

It is commonly used to display additional information, request user input, or confirm actions in web and mobile applications.

## Usage

The component comes in two variants: `edit` and `add` which you can use to alter the look and feel of the modal.
